Design a modern, premium SaaS web application called "Viral Pics", a football prediction analytics platform. The UI should feel fast, clean, data-driven, and trustworthy — not flashy like a gambling site, but like a professional analytics dashboard.

STYLE & DESIGN SYSTEM:
- Use a dark theme with a charcoal background (#0F172A or similar)
- Primary color: emerald green (#10B981) for buttons, highlights, and key actions
- Secondary accent: soft cyan/blue (#22D3EE) for data visualization and insights
- Use muted gray tones for borders, cards, and secondary text
- Add subtle gradients and soft shadows for depth
- Typography: modern sans-serif (Inter or similar), clean and readable
- UI should feel minimal, responsive, and structured
- Use card-based layout with clear spacing and hierarchy

Sections:
1. Hero Section:
- Headline: “Smarter Football Predictions Powered by Data”
- Subtext explaining automated predictions and data-driven insights

- Background: subtle gradient with data/graph visual feel

2. Features Section:
- Cards explaining:
  - Automated Predictions
